What are WebSockets?

  • A communication protocol between browsers and servers
  • Useful for creating collaborative applications, games, and chat
  • Easy to use with socket.io
  • Pretty much the coolest thing ever

What is Node.js?

  • Server-side javascript
  • Using chrome's javacsript engine
  • As fast as compiled code
  • More fun than PHP

What is Arduino?

  • A programmable development board
  • Useful for collecting data from sensors, controlling motors, and making robots
  • Many 3D printers are made with Arduino boards

Who uses this stuff?

  • Node is used by Walmart, Microsoft, Paypal, and The New York Times to name a few
  • WebSockets are used by Stack Overflow, LinkedIn, and of course Trello
